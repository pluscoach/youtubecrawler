# 로컬 서버 실행 가이드

## 방법 1: 배치 파일 (가장 쉬움)

### 서버 시작
`start.bat` 더블클릭
- 백엔드 + 프론트엔드 자동 실행
- 브라우저 자동 열림

### 서버 종료
`stop.bat` 더블클릭

---

## 방법 2: 수동 실행

### 1. 백엔드 실행
```bash
cd backend && uvicorn app.main:app --reload
```

### 2. 프론트엔드 실행 (새 터미널)
```bash
cd frontend && npm run dev
```

### 3. 브라우저 열기
http://localhost:3000

---

## Claude Code에게 요청 시
"백엔드, 프론트엔드 둘 다 실행하고 localhost:3000 브라우저로 열어줘"
